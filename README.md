# Galleria Principii

Greenfield TypeScript full-stack workspace.

## Install

```sh
npm install
```

## Development

```sh
npm run dev
npm run dev:api
npm run dev:web
```

## Test

```sh
npm test
npm run test:api
npm run test:web
npm run test:e2e
```

## Build

```sh
npm run build
```

## Verify

```sh
npm run verify
```

## Deployment

See [`docs/DEPLOYMENT.md`](docs/DEPLOYMENT.md) for the production deployment runbook (Aliyun ECS + Docker Compose + nginx + Litestream→OSS backups).
